Core Direction

Product Feel

Compact, inspectable, developer-first, and low-friction. The app should look like a focused tool, not a marketing surface.

Hierarchy

Safety state, vault, type, title, saved/used metadata, preview, and notes. Notes carry the product value.

Accent Use

The yellow primary color is for decisive actions and selected controls. Text fields use gray focus borders.

Spacing Rules

Panel Content 16px 32px 92px

Main scroll area padding. Bottom padding leaves room for fixed search.

Card Padding 16px

Default card interior. Compact cards reduce to 13px.

Card Gap 14px

Vertical rhythm between repeated clips. Compact cards use 12px.

Header Gap 10px

Gap between tag row and action buttons inside a card header.

Tag Gap 7px

Horizontal spacing between safety, vault, type, and metadata tags.

Note Gap 6px / 8px

6px between note rows, 8px between note label/value and edit action.

Field Padding 0 12px

Single-line input padding. Textareas use 10px 11px or 15px for code editing.

Radius Scale 10px / 14px / 18px

Controls 10px, cards 14px, panel 18px. Tags stay fully rounded.
